
# Formula 1 API Dashboard

## 🌐 Live Demo

https://formula1-api-dashboard.onrender.com/

---

## 📌 About

This project is built as part of my learning.
I created it to understand how to fetch data from an API and work with backend development using Flask.

The app fetches Formula 1 data from an external API and displays information like circuits, Races, Teams and Drivers in a simple web interface.
It also has some ineractive visualizations like pit, lap etc.

---

## 🛠️ Packages Used

* Flask
* Requests
* Gunicorn
* Plotly

---

## ▶️ Run Locally

```bash
git clone https://github.com/bhumikashaw/formula1-api-dashboard
cd your-repo-name
pip install -r requirements.txt
python app.py
```
